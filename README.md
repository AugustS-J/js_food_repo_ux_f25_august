# Food Repo v2
Front-end application on food recipes that consumes 2 API's

## Functionalities
- Search for random recipes
- See a recipe's details
- Sign up
- Log in
- See favorite recipes
- Mark recipe as favorite
- Unmark a recipe as favorite
- Log out

## Tools
JavaScript / CSS3 / HTML 5
